# RPG Module Platform (Planning Scaffold)

This repository contains an implementation-ready architecture scaffold for a full-stack RPG content publishing and campaign management platform.

## Planning Scope
- Frontend: React + Vite + TypeScript architecture with modular pages, components, hooks, services, context, and typed domain models.
- Backend: Express + TypeScript + Mongoose architecture separated by controllers, services, routes, models, middleware, config, and OpenAI-specific AI modules.
- Infrastructure: Docker placeholders for MongoDB and Meilisearch.
- Testing: Jest + ts-jest + Supertest setup placeholders and Playwright E2E scaffolding.

## Important Constraints Applied
- OpenAI-only AI provider preparation.
- No business logic implementation.
- Concise planning notes embedded in every scaffolded file.
