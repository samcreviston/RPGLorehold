Project Purpose
Fun group games that bring people together can oftentimes be hard to begin, and then even more so to maintain. This project is to simplify management of and make more accessible content for tabletop role playing games, such as D&D, and Pathfinder. There is a missing piece to the hobby and group game of easily accessible game content, places for people to share their game content, and a way to prepare/organize that content for an actual game. This website is to provide a platform for publishing, and in turn finding and organizing content for a simpler and easier game experience with friends and family.

Description
A React built Website:
        Core functionalites: Searching content, Creating content, Managing content, AI generated content
This website is going to be a hub for designing RPG content in a format known as modules, or a small set of adventures intended for a certain level of players. Users will be able to design and publish content with different “section”types, such as text, author notes, notes for the GM (Game Master running the adventure), images, stat blocks, item descriptions, etc. Once created and published, other users can search through the content published by others, view, favorite, and use the modules. Finally, users will be able to manage the content they have saved from their searches, to organize the modules in the order that they would play them, and to be ready to play/host a game.




Main Page
Intro and the core search feature to search for and filter through published RPG content

Module Creator Page
This is the core creation functionality. Content creators will be able to:
* Add and write text blocks, stat blocks, DM notes
* Add images, art work, maps
* Add encounter/chance rolls/randomizers
* Content Module details such as intended player level, settings, specific RPG game compatibility, enemies, bosses, etc.
* Connect D&D API info to specified text in their module to allow for hovering/clicking and pinning item/character stats/info.
* Make API calls with an AI module creation assistant

Campaign Manager Page
This is the core content management functionality. Users are able to save, bookmark, organize, and take progress notes on current modules/adventures for their group. They can also organize the order of modules for a full campaign and add their own notes.
